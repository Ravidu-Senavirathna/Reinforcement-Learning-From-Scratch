# Game Environment

This is a simple 2D py-game, where the player can increase the score by collecting points. 
Player can be controlled by the standard `WASD` keys. 
After collecting a point, a new point will appear in a random location in the screen.


Main files in this folder:
- main.py - Main game loop and logic.
- Constants.py - Contains config game data.
- Player.py - Contains the player class.
- Point.py - Contains the point class.
